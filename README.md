## 🌟 Proyectos Destacados

### 📧 **Aplicación de Envío de Correos Masivos Personalizados** (Junio 2025)
Desarrollo de una aplicación en **Python** para el envío masivo de correos electrónicos personalizados.  
Permite configurar mensajes en **HTML**, adjuntar archivos y enviar a múltiples destinatarios mediante servidores de Google, utilizando una plantilla **Excel** con datos para personalizar cada mensaje.  

---

### 🌐 **Servidor DDNS con BIND9 en Ubuntu 24.04** (Mayo 2025)
Implementación de un **servidor DDNS** configurado como DNS secundario dentro de la red interna de la empresa.  
Incluye archivos de zona directa y reversa, y se integra con un proxy existente mediante **BIND9** y **Ubuntu 24.04**.

---

### 🎓 **Plataforma LMS con Moodle** (Mayo 2025)
Desarrollo de una plataforma de formación online con **Moodle**, integrando chatbot, pagos mediante **Stripe** y panel administrativo.  
El resultado fue una solución formativa completa, adaptable y automatizada.

---

### ⚙️ **Programa de Tarificaciones Automatizadas** (Abril 2025)
Aplicación en **C# y .NET 6** con **API REST** para automatizar la gestión de tarifas.  
Optimiza tiempos y reduce errores en la gestión comercial mediante procesos de verificación e integración con otros sistemas.

---

### 🤖 **Aplicación de Automatización de Procesos** (Marzo 2025)
Herramienta en **Python** (.exe) que automatiza tareas web cada 4 horas, iniciando sesión, realizando clics y ejecutando acciones predefinidas.  
Diseñada para optimizar tareas repetitivas y ahorrar tiempo operativo.  

---

### 💰 **Aplicación de Facturación Automática** (Marzo 2025)
Sistema en **Python** que genera facturas en PDF a partir de archivos JSON, las envía por correo y las guarda automáticamente.  
Incluye scraper que actualiza tareas y está desplegado en **PythonAnywhere**.  

---

### 🔍 **Aplicación de Web Scraper** (Febrero 2025)
Aplicación en **Python** para extraer datos de sitios web y exportarlos en formato JSON.  
Incluye un scraper para la web de **Mercadona**, permitiendo obtener información detallada de productos.  

---

### 🧱 **Proxy Inverso con Nginx y Docker** (Enero 2025)
Implementación de un **proxy inverso** con **Nginx** para reforzar la seguridad de servidores Linux.  
Permite múltiples configuraciones y activa una página de mantenimiento durante actualizaciones.

---

### 🧩 **Gestor de Cursos** (Noviembre 2024)
Aplicación web para la gestión de cursos, con backend en **C#**, base de datos **MariaDB** y autenticación **JWT**.  
Incluye proxy inverso con **Nginx** y despliegue mediante **Docker**.

---

### 🎫 **Gestor de Tickets** (Octubre 2024)
Aplicación web para gestionar incidencias y tareas.  
Backend desarrollado con **C# y Entity Framework**, API de notificaciones por correo y sistema de logs.  
Despliegue en **Docker**.

---

### 👥 **Gestor de Usuarios** (2024)
Sistema web de administración de usuarios con **C#**, control de accesos mediante **JWT** y API REST.  
Enfocado en la seguridad y la integración con otros módulos empresariales.
